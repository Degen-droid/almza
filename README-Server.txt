ALMAZA WEB SERVER SETUP
======================

QUICK START:
1. Double-click "start-server.bat" to start the server
2. Open your browser and go to: http://127.0.0.1:8000/almaza-perfect-layout.html
3. On your phone (same Wi-Fi), go to: http://192.168.0.105:8000/almaza-perfect-layout.html

FILES:
- start-server.bat = Double-click this to start the server
- almaza-perfect-layout.html = Your main website file

TROUBLESHOOTING:
- If port 8000 is busy, change the port in start-server.bat from 8000 to 8001
- Make sure your phone is on the same Wi-Fi network as your PC
- If phone can't connect, check Windows Firewall settings

TO STOP SERVER:
- Press Ctrl+C in the command window that opens
- Or close the command window
